## 機能構成

### 書式・インライン要素

| 機能 | 操作方法 |
|------|---------|
| 太字 | Ctrl+B / `**text**` |
| 斜体 | Ctrl+I / `*text*` |
| 取り消し線 | `~~text~~` |
| インラインコード | `` `code` `` |
| ハイライト | `==text==` |
| 下線 | Ctrl+U |
| 上付き文字 | Ctrl+Shift+. / `text^sup^` |
| 下付き文字 | Ctrl+Shift+, / `text~sub~` |
| リンク挿入 | Ctrl+K / `[text](url)` |
| 書式クリア | Ctrl+\ |

### ブロック要素

| 機能 | 操作方法 |
|------|---------|
| 見出し H1〜H6 | Ctrl+1〜6 / `# `〜`###### ` |
| 引用ブロック | Ctrl+Shift+Q / `> ` |
| 順序なしリスト | Ctrl+Shift+] / `- ` |
| 順序付きリスト | Ctrl+Shift+[ / `1. ` |
| タスクリスト | `- [ ] ` |
| コードブロック（シンタックスハイライト） | Ctrl+Shift+K / ` ``` ` |
| 数式 KaTeX | `$...$`（インライン）/ `$$...$$`（ブロック） |
| Mermaid 図表 | ` ```mermaid ` |
| テーブル挿入 | Ctrl+T / スラッシュコマンド |
| 水平線 | Ctrl+Shift+- / `---` |

### テーブル操作

| 機能 | 操作方法 |
|------|---------|
| セル移動 | Tab / Shift+Tab |
| 行を下に追加 | Ctrl+Enter |
| 行を上に追加 | Ctrl+Shift+Enter |
| 行を削除 | Ctrl+Shift+Backspace |
| 列を右に追加 | Alt+Shift+Enter |
| 列を削除 | Alt+Shift+Backspace |
| 行/列の並び替え | 行ハンドル・列ヘッダーを D&D |
| 列幅リサイズ | 列境界をドラッグ |
| テキスト配置（左/中央/右） | セル右クリック → コンテキストメニュー |

### ファイル・ワークスペース管理

| 機能 | 説明 |
|------|------|
| タブ管理 | 複数ファイルをタブで同時編集。未保存状態を ● で表示 |
| ペイン分割 | 2 ペインで別ファイルを同時表示。D&D でペイン間移動可 |
| セッション復元 | 起動時に前回開いていたタブ・ファイルを自動復元 |
| ワークスペース | フォルダを開いてファイルツリーを表示。複数ワークスペースを切り替え |
| ドラッグ&ドロップ | ファイルをウィンドウにドロップして開く |
| ファイル関連付け | `.md` ファイルのダブルクリックで MarkWeave が起動 |
| 外部変更検知 | 外部プロセスによるファイル変更を検知し、競合解決ダイアログを表示 |
| ファイルエンコーディング | UTF-8 / UTF-8 BOM / Shift-JIS 対応。ステータスバーから変換可 |
| 改行コード | LF / CRLF を切り替えて保存可 |
| バックアップ | 定期自動バックアップ（世代管理） |
| クラッシュリカバリ | 異常終了後の起動時に未保存内容を復元 |

| 検索・置換 | Ctrl+H |
| ワークスペース横断検索 | ファイルパネル内の検索欄 |
| クイックオープン | Ctrl+P（ファイル名ファジー検索） |
| 行番号ジャンプ | Ctrl+G |

| ドキュメント先頭/末尾 | Ctrl+Home / Ctrl+End |

### エクスポート

| 出力形式 | 操作方法 |
|----------|---------|
| HTML（スタンドアロン） | Ctrl+Shift+E。GitHub / ドキュメント / プレゼン等のテーマを選択可。画像は Base64 埋め込みでリンク切れなし |
| PDF | Ctrl+Alt+P |
| Word (.docx) | Ctrl+Alt+W（Pandoc 必要） |
| LaTeX | メニュー → ファイル → エクスポート → LaTeX（Pandoc 必要） |
| ePub | メニュー → ファイル → エクスポート → ePub（Pandoc 必要） |
| Markdown として保存 | Ctrl+Shift+M（HTML → MD 変換） |
| HTML として保存 | Ctrl+Shift+H（MD → HTML 変換） |

エクスポートオプション: TOC 自動生成 / 数式レンダリング / 図表レンダリング を個別にトグル可。

### 表示・レイアウト

| 機能 | 説明 |
|------|------|
| フォーカスモード（F8） | 現在の段落のみ強調し、他の段落を薄く表示 |
| タイプライターモード（F9） | カーソル行を常に画面中央に固定 |
| Zen モード（F11） | フルスクリーン + UI 非表示で執筆に集中。ホバーでツールバー一時表示 |
| WYSIWYG / ソース切替 | Ctrl+/ でいつでも切り替え |
| テーマ | ライト / ダーク / GitHub / ドキュメント等 8 種。設定 → 外観から選択 |
| フォント | OS インストール済みフォントから選択。リガチャ ON/OFF 対応 |
| ズーム | Ctrl+= / Ctrl+- / Ctrl+0 |
| フローティング目次 | Ctrl+Shift+T で右サイドに固定表示 |

### 執筆体験強化

| 機能 | 操作方法 |
|------|---------|
| スラッシュコマンド | 行頭で `/` を入力してブロック・テンプレートを挿入 |
| スニペット | 設定 → スニペットで登録 → スラッシュコマンドで呼び出し |
| ポモドーロタイマー | ステータスバーのタイマーアイコン（25 分集中 + 5 分休憩） |
| ワードスプリント | ステータスバーから開始（時間制限付き目標文字数モード） |
| 文書統計 | 文字数・単語数・読了時間・漢字率・可読性スコア |
| 環境音（アンビエント） | Zen モード中にホワイトノイズ / 雨音 / カフェ音等 4 種 |
| タイプライター打鍵音 | 設定 → 執筆（mechanical / soft / typewriter の 3 スタイル） |
| スマートペースト | Web ページからのコピーを自動で Markdown に変換。ask モードで確認 UI |
| YAML Front Matter | エディタ上部の専用パネルで Key-Value 編集 |
| デイリーノート作成 | Ctrl+Alt+D で今日の日付ノートを作成 |
| テキスト整形コマンド | 空白除去・全角/半角変換・行ソート・重複行削除 |
| 単語補完 | Ctrl+Space で文書内の単語候補を表示 |
| スマートクォーテーション | 入力中にクォートを自動で対応する記号に変換 |

### AI 機能（BYOK 設定後に利用可能）

設定 → AI → Claude API キーを入力してから有効になります。

| 機能 | 説明 |
|------|------|
| AI コピー | 見出し補正・コードブロック言語タグ付け・過剰空白除去等を自動適用してクリップボードにコピー |
| AI テンプレート | ブログ・コード解説・コードレビュー・要約・CoT・会議メモ等 6 種の組み込みテンプレート |
| カスタムテンプレート | 独自テンプレートを作成・保存・管理 |
| RTICCO 構造診断 | プロンプトの構造的問題を検出してサイドパネルに表示 |
| **AI インライン編集** | テキスト選択 → 指示入力 → AI がストリーミングで書き換え候補を表示 → diff プレビュー → Accept/Reject |
| **AI 定型操作** | 右クリックから校正・リライト・要約・翻訳を直接実行（組み込み 5 テンプレート） |
| **AI 続きを書く** | カーソル位置から文脈を踏まえた続きの段落をストリーミング生成 |
| **参考資料コンテキスト** | ファイルを指定して AI に参考情報として送信。コンテキスト予算バーで使用量を可視化 |
| **AI 編集テンプレート** | ペルソナ / タスク / 制約 / 出力形式の 4 フィールド構造。カスタムテンプレートをローカル保存可 |

### 設定から有効化できる機能

初期状態では非表示で、設定から有効化できます。

| 機能 | 有効化場所 | 説明 |
|------|-----------|------|
| Git パネル | 設定 → 表示 | ステージング・コミット UI とエディタガター差分インジケーター |
| Wikiリンク（`[[ファイル名]]`） | 設定 → エディタ | ファイル間リンクとオートコンプリート |
| バックリンクパネル | 設定 → 表示 | 現在のファイルにリンクしている文書の一覧 |
| グラフビュー | 設定 → 表示 | Wikiリンクの繋がりを可視化 |
| タグビュー | 設定 → 表示 | YAML FM の tags を横断収集・フィルタ |
| SQLite メタデータクエリ | 設定 → 拡張機能 | ` ```query ``` ` ブロックでワークスペースをクエリ |
| Markdown Lint パネル | 設定 → 表示 | 見出しレベル飛び・リンク切れ等を検査 |
| 文章スタイル Lint（Prose Lint） | 設定 → 表示 | 文の長さ・文体混在・冗長表現・助詞の重複を検出し修正案を提示。メニュー: 表示 → Lint パネル（Ctrl+Shift+8） |
| HTML メタデータ編集パネル | 設定 → 表示 | `<title>` / `<meta>` / CSS・JS リンク管理（HTML ファイル限定） |

### HTML 編集モード

`.html` ファイルを開くと自動的に HTML モードへ切り替わります。

| 機能 | 説明 |
|------|------|
| WYSIWYG モード | h1〜h6 / p / ul / ol / table / div / セマンティック要素をビジュアル編集 |
| ソースモード | シンタックスハイライト + タグ補完付き HTML コード編集 |
| スプリットモード | 左にソース、右にプレビューをスクロール同期で表示 |
| HTML 専用ツールバー | div・セマンティック要素・テキスト色・背景色・フォントサイズ・テキスト配置等 |

## 今後実装予定の機能

### v1 リリース時に対応

| 機能 | 概要 |
|------|------|
| GitHub Releases 自動アップロード | CI でビルドアーティファクトを自動公開 |
| バージョン一括更新スクリプト | `package.json` / `Cargo.toml` / `tauri.conf.json` の同期更新 |
| Windows Authenticode 署名 | 正式販売時に対応（現状 SmartScreen 警告あり） |

### v1.5 以降

| 機能 | 概要 |
|------|------|
| Zenn / Qiita 向けエクスポート最適化 | 各プラットフォームの Markdown 仕様に合わせた出力 |
| Ghost Admin API 連携 | Ghost サイトへの記事直接公開 |
| 英語 UI | i18n 対応（英語辞書の本格整備） |

### 長期検討

| 機能 | 概要 |
|------|------|
| Markdown 拡張記法 | 脚注・カスタムコンテナ（`:::warning`）・定義リスト |
| アクセシビリティ完全準拠 | WCAG 2.1 AA（スクリーンリーダー対応） |
| macOS 対応 | 開発者が Mac 入手後に対応 |
| Android / iOS 対応 | 将来フェーズ |
