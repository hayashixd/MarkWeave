# 開発ロードマップ

> 詳細な実装計画と優先度の一覧。

## Phase 1: MVP（最小実用製品）

基本的なWYSIWYG編集が動作する状態を目指す。

### セットアップ

- [ ] Vite + React + TypeScript プロジェクト初期化
- [ ] ESLint / Prettier 設定
- [ ] Vitest テスト環境
- [ ] Playwright E2Eテスト環境

### コア機能

- [ ] ProseMirror 基本セットアップ
- [ ] マークダウンパーサ統合（remark）
- [ ] ASTシリアライザ（remark-stringify）
- [ ] ファイル読み書き（File System Access API）

### WYSIWYG要素

- [ ] 見出し（H1〜H6）
- [ ] 段落
- [ ] 太字・斜体・取り消し線
- [ ] インラインコード
- [ ] リンク
- [ ] 引用ブロック
- [ ] 順序なし・順序付きリスト
- [ ] タスクリスト
- [ ] コードブロック（シンタックスハイライト）
- [ ] 水平線
- [ ] ソースモード切替（Ctrl+/）

### オートフォーマット

- [ ] `# ` → 見出し変換
- [ ] `- ` → リスト変換
- [ ] `> ` → 引用変換
- [ ] ` ``` ` → コードブロック変換

### キーボードショートカット

- [ ] Ctrl+B（太字）
- [ ] Ctrl+I（斜体）
- [ ] Ctrl+K（リンク）
- [ ] Ctrl+Z / Ctrl+Shift+Z（Undo/Redo）
- [ ] Ctrl+S（保存）
- [ ] Ctrl+1〜6（見出しレベル）
