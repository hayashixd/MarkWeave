## アップデートポリシー

- アプリ起動時にバックグラウンドでアップデートを自動チェック
- ダウンロード・適用はユーザーが確認してから実行
- 署名済みパッケージのみインストール（改ざん検出あり）
- v1 系の間は設定ファイルの後方互換性を維持します（起動時に自動マイグレーション）

## 既知の制限事項

| 制限 | 詳細 |
|------|------|
| Windows SmartScreen 警告 | 正式版リリースまでコード署名なし。「詳細情報」→「実行」で起動できます |
| macOS 非対応 | 動作確認ができないため現時点では非対応 |
| クラウド同期なし | OS 標準のファイル同期（iCloud / Dropbox / OneDrive 等）を使ってください |
| Pandoc 必要 | Word / LaTeX / ePub エクスポートには別途 Pandoc のインストールが必要です |
| 大規模ファイル | 3MB 超のファイルは自動的にソースモードに切り替わります |

## サポート

バグ報告・機能要望は **GitHub Issues** で受け付けています。

**バグ報告の際に書いていただくと助かる情報:**
- 再現手順（どう操作したらなったか）
- OS とバージョン（例: Windows 11 22H2）
- MarkWeave のバージョン（ヘルプ → バージョン情報）
- エラーメッセージ（あれば）

**機能要望の際:**
- どんな場面で困っているか（ユースケース）を書いていただけると検討しやすいです

**返答目安:** 個人開発のため、数日かかる場合があります。

---

## データとプライバシー

- すべてのファイルはローカルに保存されます
- 外部サーバーへの送信は一切ありません
- テレメトリー・クラッシュレポートは収集しません
- **AI 機能を使用する場合:** テキストは自分で設定した Claude API キーを通じて Anthropic のサーバーに送信されます。MarkWeave のサーバーは経由しません
