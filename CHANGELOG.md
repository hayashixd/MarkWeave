# Changelog

All notable changes to MarkWeave will be documented in this file.

## [0.9.17] - 2026-03-19

### 追加
- ブロックドラッグハンドル：ブロック要素の左端にドラッグハンドルが表示され、ドラッグ＆ドロップで並び替え可能
- 外部変更検知ダイアログ：Git・Dropbox 等の外部ツールがファイルを変更した際、「エディタ内容を保持」「ディスクから再読み込み」を選択するダイアログを表示
- Zenn/Qiita UX 改善：プロファイル UI、画像ワークフロー改善、co-editing 対応

### 修正
- `.md` ファイルのダブルクリック起動時にファイルが開かれない競合状態を修正

## [0.9.0] - 2026-03-15

### 追加
- WYSIWYG Markdown エディタ本体（TipTap ベース）
- アウトラインパネル（サイドバーデフォルト表示）
- YAML Front Matter エディタ（CodeMirror）
- ファイルツリー（ワークスペース管理）
- タブ管理（複数ファイルの同時編集）
- エクスポート機能（HTML / PDF / Word / EPUB / Markdown）
- Zen モード / フォーカスモード / タイプライターモード
- ポモドーロタイマー / ワードスプリント
- 文書統計（文字数・読了時間・可読性スコア）
- テーマ切り替え（ライト / ダーク）
- 検索・置換
- IME（日本語入力）対応
- ローカルファイルファースト設計（クラウド同期なし）
- Windows / Linux 対応

### 注意
- Windows 版は Authenticode 署名なし。インストール時に SmartScreen 警告が表示される場合があります。「詳細情報 → 実行」で回避できます。
- macOS 版は現在非対応です。
