# MarkWeave クイックスタートガイド

Markdown で記事を書いて Zenn・Qiita・dev.to に公開するための、はじめの一歩ガイドです。

> **Zen モード・AI 機能・ワークスペース・ポモドーロ・全ショートカットなど上級機能は** → [advanced.md](./advanced.md)

---

## 1. はじめる（3ステップ）

### ステップ 1: ファイルを開く

`Ctrl+O` またはメニュー「ファイル → 開く...」で `.md` ファイルを選択します。
新規作成は「ファイル → 新規ファイル」（`Ctrl+N`）。

以前開いたファイルはメニュー「ファイル → 最近使ったファイル...」から素早く開けます。

### ステップ 2: 記事を書く

MarkWeave は **WYSIWYG エディタ**です。Markdown の記法を入力すると自動でレンダリングされます。

| 操作 | 方法 |
|------|------|
| 見出し | `# ` `## ` `### ` を入力（H1〜H6） |
| 太字 | `Ctrl+B` またはテキスト選択後にツールバー |
| イタリック | `Ctrl+I` |
| コード（インライン） | バッククォートで囲む |
| コードブロック | ` ``` ` + 言語名（例: ` ```typescript `） |
| リスト | `- ` または `1. ` を入力 |
| 引用 | `> ` を入力 |
| スラッシュコマンド | `/` を入力してコマンド選択 |

**表示モードの切り替え:**

- **WYSIWYG モード**（デフォルト）: 書きながら整形表示
- **ソースモード**: Markdown ソースを直接編集（`Ctrl+/`）
- **スプリットモード**: 編集とプレビューを並列表示（`Ctrl+\`）

### ステップ 3: 保存・エクスポート

- 上書き保存: `Ctrl+S`
- 名前を付けて保存: `Ctrl+Shift+S`
- Zenn / Qiita 向け Markdown エクスポート: メニュー「ファイル → エクスポート → HTML...」

## 4. テーブル

テーブルは GUI で操作できます（パイプ記法を直接書く必要はありません）。

- **挿入**: `/table` と入力してスラッシュコマンドから選択、またはツールバー
- **セル移動**: `Tab` / `Shift+Tab`
- **行・列の追加/削除**: セル右クリックメニュー
- **並び替え**: 行・列のドラッグ&ドロップ
- **列幅**: ドラッグでリサイズ

## 6. YAML Front Matter（記事メタデータ）

ファイル先頭の `---` ブロックを GUI で編集できます。
Zenn・Qiita のタイトル・タグ・公開設定の管理に便利です。

```yaml
---
title: "TypeScript で型安全な API クライアントを作る"
emoji: "🛡️"
type: "tech"
topics: ["typescript", "api"]
published: false
---
```

---

## 7. エクスポート

メニュー「ファイル → エクスポート」から形式を選択:

| 形式 | 用途 |
|------|------|
| **HTML** | スタンドアロン HTML（画像 Base64 埋め込み済み） |
| **PDF** | 印刷・配布用 |
| **Markdown として保存** | Zenn / Qiita / dev.to 投稿用（`Ctrl+Shift+M`） |
| Word / EPUB | Pandoc 連携（Pandoc 要インストール） |

---

## 8. 複数ファイルをタブで開く

複数の `.md` ファイルをタブで同時に開いて編集できます。
タブをクリックして切り替え、`×` ボタンで閉じます。

## 10. FAQ

**Q. オフラインでも使えますか？**
はい。MarkWeave はローカルファイル中心で動作します。インターネット接続は不要です（AI 機能を除く）。

**Q. 既存の Markdown ファイルをそのまま開けますか？**
はい。既存の `.md` ファイルをそのまま開いて編集できます。

**Q. Markdown 記法を知らなくても使えますか？**
はい。WYSIWYG モードではツールバーで操作できます。

**Q. Windows で SmartScreen の警告が出ます。**
現在 Authenticode 署名がないため警告が表示されます。「詳細情報 → 実行」で回避できます。

**Q. macOS に対応していますか？**
現在は Windows・Linux のみ対応です。macOS は将来対応予定です。

**Q. ファイルが外部ツール（Dropbox 等）で変更されたらどうなりますか？**
自動上書きはしません。「エディタの内容を保持」か「ディスクから再読み込み」かを選択できます。
