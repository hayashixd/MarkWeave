# ユーザーマニュアル（公式公開向け）

このマニュアルは、MarkWeave を初めて利用するユーザー向けの公開版ガイドです。
「まず使い始める」「主要機能を使う」「困ったときに確認する」を最短で辿れる構成にしています。

---

## 1. はじめに

### 1.1 このアプリでできること

- WYSIWYG で Markdown を直感的に編集
- Markdown / HTML の相互変換
- テーブル・数式・Mermaid などのリッチ要素編集
- AI コピー / AI テンプレートによる AI 活用支援

### 1.2 対応環境

- Windows / macOS / Linux（デスクトップアプリ）

---

## 2. クイックスタート

### 2.1 ファイルを開く

1. メニューまたはショートカットで「ファイルを開く」を実行
2. 編集対象の `.md` または `.html` ファイルを選択
3. エディタに内容が表示されたら編集開始

### 2.2 保存する

- 上書き保存: `Ctrl+S`
- 名前を付けて保存: `Ctrl+Shift+S`

### 2.3 表示モードを切り替える

- WYSIWYG モード: 直感的な編集
- ソースモード: Markdown / HTML ソースを直接編集
- スプリットモード: 編集結果を並行確認

---

## 3. 主要機能の使い方

### 3.1 見出し・リスト・引用・コード

- ツールバーまたはショートカットでブロック形式を変更
- リストは箇条書き・番号付き・タスクリストに対応
- コードブロックは言語指定とシンタックスハイライトに対応

### 3.2 テーブル編集（Excel ライク）

- `Tab` / `Shift+Tab` でセル移動
- 行・列の追加 / 削除
- 行・列のドラッグ並び替え
- 列幅リサイズ

### 3.3 画像・数式・Mermaid

- 画像のドラッグ&ドロップ / クリップボード貼り付け
- 数式（インライン / ブロック）
- Mermaid 記法の図表レンダリング

### 3.4 AI 機能

- **AIコピー**: ドキュメントを AI 入力向けに最適化してコピー
- **AIテンプレート**: 用途別テンプレートからプロンプト作成

---

## 4. ファイル / ワークスペース管理

- フォルダをワークスペースとして開く
- ファイルツリーで作成 / 削除 / リネーム
- 外部変更があったファイルの通知

---

## 5. よく使うショートカット

- 開く: `Ctrl+O`
- 保存: `Ctrl+S`
- 名前を付けて保存: `Ctrl+Shift+S`
- 検索: `Ctrl+F`
- 検索・置換: `Ctrl+H`
- クイックオープン: `Ctrl+P`
- 行番号ジャンプ: `Ctrl+G`

---

## 6. FAQ

### Q1. Markdown 記法を知らなくても使えますか？

はい。WYSIWYG モードでは、記法を強く意識せずに編集できます。

### Q2. オフラインでも使えますか？

はい。ローカルファイル中心で動作するため、オフライン利用が可能です。

### Q3. 既存の Markdown / HTML ファイルは開けますか？

はい。既存ファイルをそのまま開いて編集できます。

### Q4. 大きいファイルは扱えますか？

はい。大容量ファイル向けの最適化（表示モード切替など）を実装しています。
