# ユーザーマニュアル（公式公開向け）

このマニュアルは、MarkWeave を初めて利用するユーザー向けの公開版ガイドです。
「まず使い始める」「主要機能を使う」「困ったときに確認する」を最短で辿れる構成にしています。

---

## 1. はじめに

### 1.1 このアプリでできること

- WYSIWYG で Markdown を直感的に編集（Typora ライク）
- 見出し / リスト / コード / テーブル / 数式 / Mermaid 図表などのリッチ要素編集
- HTML / PDF / Word / EPUB / Markdown へのエクスポート
- Zen モード・フォーカスモード・タイプライターモードでの集中執筆
- AI コピー / AI テンプレートによる AI 活用支援

### 1.2 対応環境

- Windows（v0.9.0〜）
- Linux（v0.9.0〜）
- ※ macOS は現在非対応です。

---

## 2. クイックスタート

### 2.1 ファイルを開く

1. メニュー「ファイル → 開く」またはショートカット `Ctrl+O` を実行
2. 編集対象の `.md` または `.html` ファイルを選択
3. エディタに内容が表示されたら編集開始

### 2.2 保存する

- 上書き保存: `Ctrl+S`
- 名前を付けて保存: `Ctrl+Shift+S`

### 2.3 表示モードを切り替える

- **WYSIWYG モード**: 記法を意識せず直感的に編集（Typora 式）
- **ソースモード**: Markdown ソースを直接編集
- **スプリットモード**: 編集とプレビューを並行表示

---

## 3. 主要機能の使い方

### 3.1 見出し・リスト・引用・コード

- ツールバーまたはショートカットでブロック形式を変更
- リストは箇条書き・番号付き・タスクリストに対応
- コードブロックは言語指定とシンタックスハイライトに対応
- `# ` を入力すると自動的に見出しへ変換（入力ルール）

### 3.2 テーブル編集（Excel ライク）

- `Tab` / `Shift+Tab` でセル移動
- 行・列の追加 / 削除
- 行・列のドラッグ並び替え
- 列幅リサイズ・セル配置変更

### 3.3 画像・数式・Mermaid

- 画像のドラッグ&ドロップ / クリップボード貼り付け
- 数式（インライン / ブロック）：KaTeX 形式
- Mermaid 記法の図表リアルタイムレンダリング

### 3.4 YAML Front Matter

- ファイル先頭の YAML フロントマターを GUI で編集
- 記事タイトル・タグ・公開日などのメタデータを管理

### 3.5 タブ管理

- 複数ファイルをタブで同時に開いて編集
- タブをクリックして切り替え、`×` ボタンで閉じる

### 3.6 執筆支援モード

| モード | 説明 | 呼び出し方 |
|--------|------|----------|
| Zen モード | サイドバー・メニューを非表示にして執筆に集中 | メニュー → 表示 → Zen モード |
| フォーカスモード | カーソル周辺のみハイライト | メニュー → 表示 → フォーカスモード |
| タイプライターモード | カーソル行を常に画面中央に配置 | メニュー → 表示 → タイプライターモード |
| ポモドーロタイマー | 作業時間を区切って管理（25分+休憩） | メニュー → ツール → ポモドーロ |
| ワードスプリント | 時間制限付き執筆チャレンジ | メニュー → ツール → ワードスプリント |

### 3.7 文書統計

- ステータスバーで文字数・読了時間・可読性スコアをリアルタイム表示

### 3.8 テーマ切り替え

- メニュー → 表示 → テーマ からテーマを変更（ライト / ダーク 等）

### 3.9 AI 機能

- **AIコピー**: ドキュメントを AI 入力向けに最適化してコピー（ツールバーの [AIコピー] ボタン）
- **AIテンプレート**: 用途別テンプレートからプロンプト作成（サイドバーの [AI テンプレート] タブ）
- ※ 利用には Claude API キーの設定が必要です（メニュー → 設定 → AI）

## 4. ファイル / ワークスペース管理

- フォルダをワークスペースとして開く（メニュー → ファイル → フォルダを開く）
- ファイルツリーで作成 / 削除 / リネーム
- 外部変更があったファイルの通知（Dropbox・OneDrive 等で変更された場合）
- クイックオープン（`Ctrl+P`）でファイルを素早く検索・開く

---

## 5. エクスポート

メニュー → ファイル → エクスポート から形式を選択:

| 形式 | 概要 |
|------|------|
| HTML | スタンドアロン HTML（画像 Base64 埋め込み済み） |
| PDF | 印刷品質の PDF |
| Word | .docx 形式（Pandoc 連携） |
| EPUB | 電子書籍形式（Pandoc 連携） |
| Markdown | 最適化済み Markdown |

## 7. FAQ

### Q1. Markdown 記法を知らなくても使えますか？

はい。WYSIWYG モードでは、記法を強く意識せずに編集できます。

### Q2. オフラインでも使えますか？

はい。ローカルファイル中心で動作するため、オフライン利用が可能です。

### Q3. 既存の Markdown / HTML ファイルは開けますか？

はい。既存ファイルをそのまま開いて編集できます。

### Q4. macOS に対応していますか？

現在は Windows と Linux のみ対応しています。macOS は将来対応予定です。

### Q5. Windows でインストール時に SmartScreen 警告が出ます。

現在 Authenticode 署名がないため、SmartScreen 警告が表示される場合があります。
「詳細情報 → 実行」をクリックすることで回避できます。

### Q6. 大きいファイルは扱えますか？

はい。大容量ファイル向けの最適化（3MB 超でソースモード自動切替など）を実装しています。

### Q7. AI 機能を使うには何が必要ですか？

Claude の API キーが必要です。メニュー → 設定 → AI で API キーを設定してください。
AI サービス利用料は API キーに応じた Anthropic の料金体系に準じます。
