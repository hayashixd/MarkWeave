## 1. プロダクト概要

- 製品名: **Markdown / HTML Editor - AI時代のWYSIWYGエディタ**
- コンセプト: Typora ライクな操作感で、Markdown / HTML を直感的に編集
- 提供価値:
  - Markdown記法を意識しないリアルタイム編集
  - Markdown ↔ HTML の相互変換
  - AI活用を前提とした編集支援（AIコピー / テンプレート）

公開用コピー案:

> 書く・整える・AIに渡すを、ひとつのエディタで。
> Typoraライクな直感操作で、Markdown / HTML をシームレスに編集できます。

---

## 2. 主要機能（公開用）

### 2.1 WYSIWYG編集

- Typora式インラインレンダリング
- 見出し / リスト / 引用 / コード / テーブル / 数式 / Mermaid をサポート
- ソースモード / スプリットモードを含む複数表示モード

### 2.2 テーブル編集（Excelライク）

- セル移動（Tab / Shift+Tab）
- 行・列の追加/削除
- 行・列のドラッグ並び替え
- 列幅リサイズ・配置変更

### 2.3 ファイル・ワークスペース

- ファイルを開く / 保存 / 名前を付けて保存
- フォルダをワークスペースとして開く
- ファイルツリー表示、作成/削除/リネーム
- 外部変更検知

### 2.4 AI連携

- **AIコピー**: ドキュメントをAI向けに自動最適化してクリップボードへコピー
- **AIテンプレート**: ブログ構成、コード解説、レビュー依頼、要約などのテンプレート提供

### 2.5 画像編集

- 画像アノテーション（矩形、楕円、矢印、フリーハンド、テキスト、モザイク等）
- Undo 対応

## 3. 対応プラットフォーム / 技術基盤

- デスクトップ: Windows / macOS / Linux（Tauri 2.0）
- フロントエンド: React + TypeScript + Vite
- バックエンド: Rust

※ 公式サイトには「利用者価値」を優先して記載し、内部実装詳細は最小限にする。

## 4. 導入情報（公開用）

- Node.js 20+
- pnpm 9+
- Rust 1.77.2+
- Tauri 前提ライブラリ（OS依存）

- `pnpm install`
- `pnpm dev`（フロントのみ）

### 8.1 想定ユーザー / ユースケース

- 技術文書を書くエンジニア・テクニカルライター
- 記事作成を行うブロガー / コンテンツ制作者

- **Markdown / HTML 一体運用**: 片方に閉じない編集フロー
- **大規模ファイル対応**: サイズに応じた表示モード最適化

公開用コピー案:

> 下書きから最終出力まで、Markdown と HTML を往復しながら、
> AI 活用を前提にした実務フローを一つのアプリで完結できます。
