## 1. プロダクト概要

- 製品名: **MarkWeave**
- コンセプト: Write. Polish. Publish. — Webで文章を書いて公開する人のための、ローカルファーストWYSIWYGエディタ
- ターゲット: Markdown で技術記事・ブログ記事を書いて、Zenn / Qiita / dev.to / Hashnode に公開するエンジニア・テクニカルライター
- 提供価値:
  - Typora ライクな書き心地で Markdown をリアルタイム編集
  - プラットフォーム向け公開ワークフロー（Zenn / Qiita 最適化エクスポート）
  - ローカルファースト設計（クラウド不要・ベンダーロックインなし）
  - AI 活用支援（AIコピー・AIテンプレート）

公開用コピー案:

> 書く。整える。公開する。
> Typora ライクな直感操作で Markdown を書いて、技術ブログプラットフォームへワンクリック公開。
> ファイルはすべてローカルに。クラウド不要。

## 2. 主要機能（公開用）

### 2.1 WYSIWYG編集

- Typora 式インラインレンダリング（フォーカス時にソース表示、非フォーカス時にレンダリング）
- 見出し / リスト / 引用 / コード / テーブル / 数式（KaTeX）/ Mermaid 図表をサポート
- 表示モード: WYSIWYG / ソースモード / スプリットモード（サイドバイサイド）
- YAML Front Matter エディタ（記事メタデータの GUI 編集）
- タブで複数ファイルを同時編集

### 2.2 テーブル編集（Excelライク）

- セル移動（Tab / Shift+Tab）
- 行・列の追加 / 削除
- 行・列のドラッグ並び替え
- 列幅リサイズ・配置変更

### 2.3 ファイル・ワークスペース

- ファイルを開く / 保存 / 名前を付けて保存
- フォルダをワークスペースとして開く
- ファイルツリー表示、作成 / 削除 / リネーム
- 外部変更検知（Dropbox・OneDrive 等との安全な共存）

### 2.4 執筆支援モード

- **Zen モード**: メニュー・サイドバーを非表示にして執筆に集中
- **フォーカスモード**: カーソル周辺のみハイライト
- **タイプライターモード**: カーソル行を常に画面中央に配置
- **ポモドーロタイマー**: 作業時間の区切り管理
- **ワードスプリント**: 時間制限付き執筆チャレンジ

### 2.5 文書統計・テーマ

- 文字数 / 読了時間 / 可読性スコアのリアルタイム表示
- テーマ切り替え（ライト / ダーク / その他）

### 2.6 エクスポート

- HTML（スタンドアロン、画像 Base64 埋め込み）
- PDF
- Word / EPUB（Pandoc 連携）
- Markdown（プラットフォーム向け最適化）

### 2.7 AI機能

- **AIコピー**: ドキュメントをAI入力向けに自動最適化してクリップボードへコピー
- **AIテンプレート**: ブログ構成・コード解説・コードレビュー依頼・要約などのテンプレート提供

### 2.8 マニュアル（ユーザーガイド）

- 公式マニュアルをサイト上で参照可能（`doc-public/manuals/user-manual.md` を公開原稿として利用）
- 機能ごとの操作手順とスクリーンショットを掲載
- アプリ更新に合わせて内容を継続更新
