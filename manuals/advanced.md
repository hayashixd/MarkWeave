# 高度な機能ガイド

基本的な使い方は [user-manual.md](./user-manual.md) を参照してください。
ここでは上級者向けの機能を説明します。

---

## 1. 執筆支援モード

### Zen モード

メニュー「表示 → Zen モード」（`F11`）

サイドバー・ツールバー・ステータスバーをすべて非表示にして執筆に集中するモードです。

### フォーカスモード

メニュー「表示 → フォーカスモード」

現在編集中の段落周辺のみをハイライトし、前後の文章をグレーアウトします。

### タイプライターモード

メニュー「表示 → タイプライターモード」

カーソル行を常に画面中央に配置します。長文執筆に適しています。

---

## 2. 執筆ツール

### ポモドーロタイマー

メニュー「ツール → ポモドーロ」

25分作業 + 5分休憩のサイクルで執筆時間を管理します。

### ワードスプリント

メニュー「ツール → ワードスプリント」

時間制限（例: 10分）で目標文字数を達成するチャレンジモードです。

---

## 3. 文書統計

ステータスバーに以下がリアルタイムで表示されます:

- 文字数 / 単語数
- 推定読了時間
- 可読性スコア

---

## 4. テーマ切り替え

メニュー「表示 → テーマ」からライト / ダーク / その他のテーマを選択できます。

---

## 5. ワークスペース（フォルダ管理）

メニュー「ファイル → フォルダを開く」でフォルダをワークスペースとして開くと:

- 左サイドバーにファイルツリーが表示されます
- フォルダ内のファイル作成 / 削除 / リネームができます
- クイックオープン（`Ctrl+P`）でフォルダ内のファイルを素早く開けます

**外部変更の検知**: Dropbox・OneDrive 等で外部からファイルが変更された場合、自動上書きせず通知します。

## 6. AI 機能（BYOK）

MarkWeave の AI 機能は **BYOK（Bring Your Own Key）** 方式です。
自分の Claude API キーを設定して利用します（Anthropic の料金体系が適用されます）。

### セットアップ

1. メニュー「設定 → AI」を開く
2. Anthropic の Claude API キーを入力して保存

### AI コピー

ツールバーの「AI コピー」ボタンを押すと、現在のドキュメントを AI 入力向けに最適化した形式でクリップボードにコピーします。
Claude / ChatGPT などにそのまま貼り付けてレビューや改善を依頼する際に便利です。

### AI テンプレート

サイドバーの「AI テンプレート」タブから用途別のプロンプトテンプレートを選択できます:

- ブログ記事構成
- コード解説
- コードレビュー依頼
- 要約・リライト

## 8. 大規模ファイルの扱い

MarkWeave は大容量ファイルにも対応しています:

- **50KB 超**: UI ペイント後に遅延処理
- **3MB 超**: ソースモードに自動切替（WYSIWYG パースを回避して高速化）

通常の技術記事（数十 KB）では意識する必要はありません。

## 9. FAQ（上級）

**Q. AI 機能を使うには何が必要ですか？**
Claude の API キーが必要です。メニュー「設定 → AI」で設定してください。API 利用料は Anthropic の料金体系に準じます。

**Q. 大きいファイルは扱えますか？**
はい。3MB 超のファイルはソースモードに自動切替して対応します。

**Q. クラウド同期はできますか？**
MarkWeave 自体はクラウド同期機能を持ちません。iCloud・Dropbox・OneDrive などの OS 標準のファイル同期と組み合わせてご利用ください。
